Alinhamento de Texto

Como alinhar?

Não usama tags de HTML para alinhhar texto como por exemplo: "center." Devemos usa folha de estilos de CSS para alinhar o nosso texto usando por exemplo: "text-align:right;"(alem das outra propiedade "left" "center" "justify"). Assim faz alinhamento mas quando queremos indicar que novo paragrafo com espaço do inicio de paragrafo temos que usar "text-indent:20px"ou seja espaço é determinado pelo numero de pixel (px.
